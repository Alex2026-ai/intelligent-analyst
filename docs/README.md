# Public Documentation

This directory is intentionally small for the public release branch.

Included public materials:

- `IAVP_V1_PUBLIC_SPEC.md` — public protocol specification.
- `IAVP_EVIDENCE_SCHEMA_V1.md` — evidence artifact schema.
- `IAVP_VERIFICATION_WALKTHROUGH.md` — verifier walkthrough.
- `TRANSPARENCY_PUBLIC_API.md` — public transparency API notes.
- `EXTERNAL_VERIFICATION_DEMO.md` — external verification demo flow.
- `protocol/ia_attestation_manifest_v1.md` — attestation manifest reference.
- `PUBLIC_RELEASE_CHECKLIST.md` — final checklist before opening the repo.

Internal runbooks, production snapshots, legal drafts, pricing notes, sales scripts, and operator-only materials are intentionally excluded from the public branch.
